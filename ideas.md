## Idea A — "Tactical Obsidian" (Cinematic Military-Grade)

**Design Movement:** Dark Brutalism meets Cinematic Surveillance Aesthetic

**Core Principles:**
1. Absolute darkness — deep obsidian backgrounds with no softness
2. Electric cyan as the only light source — like a radar screen
3. Monospace + condensed sans-serif typography for tactical feel
4. Asymmetric grid with hard-cut diagonal sections

**Color Philosophy:**
- Background: #050A0F (near-black with blue undertone)
- Primary accent: #00E5FF (electric cyan — surveillance monitor glow)
- Secondary: #1A2535 (dark navy panels)
- Metallic: #8A9BB0 (steel gray for secondary text)
- Danger/alert: #CC2200 (red — matching original logo)
- White: #E8F4F8 (cold white for headings)

**Layout Paradigm:**
- Full-width sections with diagonal clip-path cuts between them
- Left-aligned hero text with right-side 3D shield visual
- Asymmetric card grid for services (3+2 layout)
- Sticky header with glass blur effect

**Signature Elements:**
1. Animated surveillance grid (SVG dot grid with scan-line animation)
2. Glowing cyan border-left on cards (like terminal readouts)
3. Shield emblem with rotating outer ring and glow pulse

**Interaction Philosophy:**
- Hover: cyan glow expansion + slight scale-up
- Scroll: section fade-in with upward translate
- CTA buttons: border-glow animation on hover

**Animation:**
- Hero: staggered text reveal (word by word)
- Grid background: slow scan-line sweep
- Cards: entrance from bottom with opacity fade
- Shield: continuous slow rotation of outer ring

**Typography System:**
- Display: Barlow Condensed 700 (headings, hero)
- Body: Barlow 400/500 (paragraphs)
- Accent: Space Mono (stats, labels, technical data)
- Scale: 72px hero → 48px h2 → 32px h3 → 16px body

## CHOSEN APPROACH: Idea A — "Tactical Obsidian"

**Rationale:** Perfectly aligns with the brief's requirements for cinematic, dark, 3D security aesthetic. The electric cyan surveillance aesthetic creates maximum visual impact while maintaining corporate seriousness. The diagonal sections and asymmetric layout avoid the "template" look explicitly requested to be avoided.

**Key design decisions:**
- Font: Barlow Condensed (closest to Arial but more premium/corporate)
- Primary: #00E5FF electric cyan
- Background: #050A0F obsidian
- Animated surveillance grid background
- Shield logo preserved with glow enhancement
- All text in Greek
